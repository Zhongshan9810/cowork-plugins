# 打工人 (dagong)

办公文档处理技能包 - Word、Excel、PDF、PPT 创建与编辑，内部沟通文档撰写。

## Skills

| Skill | 功能 |
|-------|------|
| docx | Word 文档创建/编辑 |
| xlsx | Excel 表格处理 |
| pdf | PDF 处理 |
| pptx | PowerPoint 演示文稿 |
| internal-comms | 内部沟通文档 |
| skill-creator | 创建新技能 |
| create-shortcut | 创建快捷方式 |

## 使用方式

这些 skills 会根据任务自动触发。例如：
- "帮我创建一个 Word 文档" → docx skill
- "处理这个 PDF" → pdf skill
- "创建一个 Excel 表格" → xlsx skill
